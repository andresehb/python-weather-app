# 🌦️ Python Weather App

A Python application that fetches and displays real-time weather data using a public API.

## 🚀 Overview
This project demonstrates how to work with external APIs, handle JSON data, and present useful information to users in a clean and simple way.

## ✨ Features
- Fetches real-time weather data from an external API  
- Parses JSON responses and extracts relevant information  
- Displays temperature, weather conditions, and location data  
- Handles basic errors (e.g., invalid city or API issues)  

## 🛠️ Tech Stack
- Python  
- Requests (HTTP library)  
- REST APIs
